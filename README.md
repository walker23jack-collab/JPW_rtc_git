# JPW_rtc
This github will include JPW's code changes for the StrategicHeart project
# Description
The project will assess the capacity for addition of an ASR well to the Dutch water supply. Water is extracted from the IJssel river (**Qint**), brought to a processing basin, then to a treatment plant (**Qtreat**). From the plant, water will either be distributed to the network (**Qdis**) or infiltrated at an ASR well (**Qinf**) for future extraction (**Qext**)
# Goals
1. Satisfy Qdem
2. Meet min and max Qint requirements
3. Maximize Qtreat
